## SYSTEM DESIGN PLATFORMS

### PaperDraw.dev
- **Best at:** Only tool combining diagramming + live simulation + chaos testing
- **Missing:** No educational content, no AI feedback, no DSA, no collab, no export
- **Price:** Free

### Excalidraw
- **Best at:** Fastest whiteboard sketching, hand-drawn aesthetic, open-source, real-time collab
- **Missing:** No simulation, no educational content, no system design components
- **Price:** Free / $6-7/user/mo

### Eraser.io
- **Best at:** Diagram-as-code, Git integration, Eraserbot auto-updates diagrams on PRs
- **Missing:** No simulation, no learning, no DSA, enterprise-focused
- **Price:** $10-25/user/mo

## DSA VISUALIZATION PLATFORMS

### VisuAlgo
- **Best at:** Broadest coverage (26 modules including NP-hard, network flow, geometry)
- **Missing:** Dated UI, no custom code, no collaboration, no mobile
- **Price:** Free

### Algorithm Visualizer
- **Best at:** Code-first — you write code, see it visualize (unique approach)
- **Missing:** Limited coverage, no quiz, community slowed
- **Price:** Free, open-source

### USFCA Galles
- **Best at:** Deepest tree/heap coverage (Fibonacci Heaps, B+ Trees, Skew Heaps)
- **Missing:** Ancient UI, no code, no explanations
- **Price:** Free

### Python Tutor
- **Best at:** THE tool for memory visualization (stack frames, pointers, heap)
- **Missing:** No algorithm animation, small programs only
- **Price:** Free

## LEARNING PLATFORMS

### NeetCode.io
- **Best at:** Best curation of LeetCode problems, pattern-based organization
- **Missing:** System design is shallow, no visualization, no mock interviews
- **Price:** $149/yr

### Educative.io
- **Best at:** Text-based interactive courses, in-browser coding, 1600+ courses
- **Missing:** No visualization, no diagramming, quality varies
- **Price:** $59/mo or $200/yr

### DesignGurus.io
- **Best at:** "Grokking" methodology, System Design Master Template, 60+ problems
- **Missing:** No visualization, no mock interviews, expensive monthly
- **Price:** $119/mo or $349/yr

### ByteByteGo
- **Best at:** BEST visual explanations in the market (Alex Xu's diagrams are industry-standard)
- **Missing:** Consumption-only, no interactivity, no coding, no practice
- **Price:** $189/yr or $499 lifetime

### Hello Interview
- **Best at:** AI tutor + human coach mock interviews, company-specific guides
- **Missing:** No visualization, mock interviews expensive ($170-289 each)
- **Price:** $35-55/mo or $210 lifetime + mock fees

### Codemia.io
- **Best at:** Iterative AI feedback loop (design → score → refine → resubmit) — closest to "LeetCode for system design"
- **Missing:** No visualization, no diagramming canvas, relatively new
- **Price:** $55-69/yr or $159 lifetime

## 10 CRITICAL MARKET GAPS (Our Competitive Advantage)

### GAP 1: Unified Platform
No platform combines DSA visualization + system design simulation + practice + AI scoring in one experience.

### GAP 2: "Run Your Architecture"
PaperDraw has simulation but zero education. ByteByteGo has education but zero interactivity. Nobody has both.

### GAP 3: Code-to-Architecture
No tool auto-generates system architecture from actual code.

### GAP 4: DSA-to-System-Design Bridge
Nobody connects "how a hash map works" → "consistent hashing at scale" → "design a distributed cache."

### GAP 5: Collaborative System Design Practice
No platform enables two people to collaboratively design with AI interviewer + scoring.

### GAP 6: Failure Mode Visualization
Nobody shows what happens when algorithms have bugs or systems lose nodes.

### GAP 7: Company-Specific Adaptive Learning
No platform dynamically adjusts learning paths based on target company.

### GAP 8: Mobile-First System Design
Almost every platform is desktop-only.

### GAP 9: Open-Source Self-Hostable Platform
No open-source platform combines system design + DSA + AI feedback. Universities would adopt immediately.

### GAP 10: Concurrency & Distributed Systems Visualization
Python Tutor does sequential brilliantly, but NOBODY visualizes concurrent threads, race conditions, Raft/Paxos, or vector clocks.
