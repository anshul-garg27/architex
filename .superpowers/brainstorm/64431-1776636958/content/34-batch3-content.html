C6

"Draw from memory" challenge · pattern as pure recall

Novel format: pattern page has a "Close your eyes, open this" button. Canvas is blank. User gets 3 minutes to drag-build the pattern from scratch — no reference, no starter. Auto-grader checks. Pass = Mastered tier bonus. Feels like a pop quiz, but voluntary.

Uses existing grading engine. Timer + "no reference" mode toggle on canvas. Spaced repetition gold — this is Anki with UML. ~6h.
C7

Side-by-side multi-language viewer · TS / Py / Java / Go / Rust columns

Current design shows code as tabs (switch one at a time). Propose: optional "compare" view showing the same pattern in 3 languages side-by-side. See how idiomatic differences express the same intent — Python's __instance__ vs Java's synchronized. Requires adding Go + Rust snippets to the 3 existing (TS/Py/Java).

UI toggle: tabs → columns. Adding Go + Rust snippets for 36 patterns ≈ 8-10h Opus. UI work ≈ 4h.
C8

Audio narration · podcast-style lesson mode

Each lesson gets a 5-8 min audio narration (Claude Opus voice-gen or human). "Play audio" button in Learn mode — canvas highlights follow narration timestamps. Perfect for commutes, bathtub, running. Turns reading into listening.

Audio production: 36 patterns × 6 min avg = 3.6 hours of content. AI voice gen (~$200 ElevenLabs) OR human narrator (~$2K). Timestamp alignment with canvas highlights = ~10h. Audio first-mover advantage in tech-education.
C9

Pattern origin stories · 1-paragraph history per pattern

"Who coined this? Why? When?" Small history card per pattern: "Erich Gamma et al. coined Observer in the 1995 Gang of Four book. Inspired by the Model-View-Controller pattern at Xerox PARC in 1979." Makes patterns feel like human inventions, not cosmic laws. Counter to the "memorize 36 abstract names" feel.

Opus writes: 36 × 5-min research each ≈ 3h content. ~2h UI. Small effort, big humanization.
