# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Architex, please report it responsibly:

1. **Do NOT** open a public GitHub issue
2. Email: security@architex.dev (or use GitHub's private vulnerability reporting)
3. Include: description, steps to reproduce, potential impact
4. We will respond within 48 hours
