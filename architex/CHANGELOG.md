# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-04-11

### Added

- **System Design Simulator**: drag-and-drop architecture builder with 30+ node types, live traffic simulation, chaos engineering, and real-time metrics dashboard
- **Algorithm Visualizer**: step-by-step animation for 26+ algorithms across sorting, graph, tree, dynamic programming, and string categories with playback controls
- **Data Structure Explorer**: interactive visualization of arrays, linked lists, hash tables, BSTs, and bloom filters
- **Distributed Systems module**: Raft consensus, consistent hashing, vector clocks, gossip protocol, CRDTs, and CAP theorem simulations
- **Networking and Protocols module**: TCP, TLS 1.3, DNS resolution, HTTP/1-2-3 comparison, WebSocket, and CORS visualizations
- **Operating Systems module**: CPU scheduling (6 algorithms), page replacement, deadlock detection, and virtual memory simulations
- **Security and Cryptography module**: OAuth 2.0 PKCE flow, JWT lifecycle, and Diffie-Hellman key exchange demonstrations
- **ML System Design module**: neural network trainer with live decision boundary visualization
- **Database Design module**: normalization, ER diagrams, indexing strategies, and query optimization tools
- **LLD Patterns module**: low-level design patterns with interactive class diagrams
- **Concurrency module**: thread scheduling, race conditions, mutex and semaphore demonstrations
- **Interview Engine**: 20+ system design challenges with timer, scoring, and auto-requirement detection
- **Design System**: shadcn/ui component library with Storybook documentation
- **Simulation engine**: traffic simulation with queuing theory, chaos engineering, and configurable failure injection
- **Canvas infrastructure**: React Flow v12 integration with custom nodes, edges, panels, minimap, and overlays
- **State management**: 6 Zustand stores (canvas, editor, UI, simulation, interview, viewport) with zundo undo/redo
- **Local persistence**: Dexie (IndexedDB) for saving diagrams and session state
- **Export system**: diagram export utilities for sharing and documentation
- **Pre-built templates**: architecture templates for common system design patterns
- **Web Workers**: off-thread computation for simulation and algorithm engines
- **Keyboard shortcuts**: comprehensive shortcut system for power users
- **Command palette**: quick-access command interface (cmdk)
- **Dark theme**: full dark mode support via next-themes
- **Storybook**: component development environment with addon-essentials and addon-interactions
- **Bundle analyzer**: @next/bundle-analyzer integration for build analysis
- **Testing infrastructure**: Vitest with Testing Library for unit and component tests
