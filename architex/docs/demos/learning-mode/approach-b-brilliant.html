How Does the Subject Notify?

The Subject maintains a list of observers. When its state changes, it calls notify() which iterates through the list and calls update() on each observer.

Check your understanding
A The Subject creates a new Observer instance
B It gets added to the Subject's internal observers list
C The Observer immediately receives all previous state changes
D notify() is called automatically for the new Observer
Correct! The observer is simply appended to the subject's list.
